== Description ==
BIT SMTP allows you to configure and send all outgoing emails via a SMTP server. This will prevent your emails from going into the junk/spam folder of the recipient.

### Features:
1. Send email using a SMTP sever
2. You can use Gmail, Yahoo, Hotmail’s,Mailgun,Aws ses,Zoho Mail SMTP server if you have an account with them
3. Securely deliver emails to your recipients.
4. Ability to specify a Reply-to email address)
5. Email Log (Coming soon)
6. Email Control (Coming soon)

== Installation ==

1. Download the plugin.
2. From the WordPress Admin Panel, click on Plugins => Add New.
3. Click on Upload, so you can directly upload your plugin zip file.
4. Use the browse button to select the plugin zip file that was downloaded, and then click on Install Now.
5. Once installed, click “Activate”.

== BIT SMTP PLUGIN USAGE ==
Once you have installed the plugin, Click BIT SMTP menu

The general section consists of the following options
1. Enable SMTP: Yes/No if option Yes plugin is enable or option is No plugin is disable (This option should always be checked “Yes”).
2. From Email Address: The email address that will be used to send emails to your recipients.
3. From Name: The name your recipients will see as part of the “from” or “sender” value when they receive your message.
4. Reply-To Email Address :  The email address that will be used to Reply-to email address to your recipients.
5. SMTP Host: Your outgoing mail server (example: smtp.gmail.com).
6. Type of Encryption: SSL/TLS.
7. SMTP Port: The port that will be used to relay outbound mail to your mail server (example: 587).
8. SMTP Authentication: No/Yes (This option should always be checked “Yes”).
9. Username: The username that you use to login to your mail server.
10. Password: The password that you use to login to your mail server.
